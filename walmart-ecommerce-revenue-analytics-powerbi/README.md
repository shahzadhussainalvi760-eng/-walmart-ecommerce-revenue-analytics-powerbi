
# 🛒 Walmart E-Commerce Revenue Intelligence & Customer Analytics Dashboard

![Dashboard Preview](assets/dashboard_preview.png)

## 📌 Project Overview

This Power BI project delivers a comprehensive **E-Commerce Revenue Intelligence 
& Customer Analytics Dashboard** built for Walmart's retail data. It features 
**7 interactive report pages** covering revenue, customer behavior, product 
performance, geographic distribution, payment trends, and time-based sales patterns.

---
